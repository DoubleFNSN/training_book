#### Example I. mean and t test

---

```
sh# R
 x<-read.table("foo1")
 y<-read.table("foo2") 
 mean(x) 
 mean(y) 
 ?t.test 
 t.test( x,y,alternative=c("greater") )
```

Or write this into a file, script.R, then  
`R CMD BATCH script.R`
